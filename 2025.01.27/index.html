element blokowy header - określa blok nagłówkowy, często nazywany blokiem banera
elelemnt blokowy main - określa blok główny
element blokowy aside, określa blok boczny
element blokowy article - określa artykuł, część dokumnetu
element blokowy nav - określenie bloku nawigacji (menu)
element blokowy footer - określa stopkę(dół) strony
element blokowy section - ogólny element blokowy zgodny html5
element blokowy div - element używany w starszych wersja html (przed html5), nie zaleca się jego stosowania w kodzie

Element blokowy nagłówka 1 stopnia "h1" - mamy 6 stopni nagłówka

element blokowy paragraf(akapit) "p"- w nim umieszczamy fragment tekstu. Działa tak jak fragment tekstu w książce, po zakończeniu akapitu tekst rozpoczyna się od nowej linii
